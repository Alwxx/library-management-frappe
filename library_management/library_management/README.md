# Library Management System (Frappe)

## Features
- Book Issue & Return
- Stock Management
- Duplicate Issue Prevention
- Fine Calculation for Late Returns

## Tech Stack
- Frappe Framework (Python)
- MariaDB
- JavaScript (Frontend)

## Modules
- Book
- Member
- Book Issue
- Book Return

## Key Logic
- Prevent issuing same book twice
- Reduce stock on issue
- Increase stock on return
- Calculate fine automatically
